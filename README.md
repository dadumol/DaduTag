Frequently Asked Questions
<details> <summary><strong>How do I add a new tag?</strong></summary>
From the Home screen, tap the + button in the Tags section. Enter a name, choose an icon and color, then save. Your tag will appear on the dashboard immediately.

</details> <details> <summary><strong>Why is my tag showing an old location?</strong></summary>
Location updates depend on your tag hardware's reporting interval and internet connectivity. If a tag hasn't updated recently, it will show the last known location with a timestamp. Check that the tag device has power and a network connection.

</details> <details> <summary><strong>How do I set up an alert?</strong></summary>
Go to the Alerts tab, tap New Alert, select the tag and rule type (zone entry/exit, offline, stationary), configure the conditions, and save. You'll receive a push notification when the rule triggers.

</details> <details> <summary><strong>Can I share a tag with someone else?</strong></summary>
Tag sharing between multiple accounts is on our roadmap for a future update. Currently, each tag is tied to one account.

</details> <details> <summary><strong>How do I delete my account and data?</strong></summary>
Go to Settings → Account → Delete Account. This permanently removes all your data from our servers. This action cannot be undone.

</details> <details> <summary><strong>The app is draining my battery — what can I do?</strong></summary>
DaduTag uses background location only when live tracking is active. If you're not actively using it, close the live map view. You can also reduce background refresh in iOS Settings → DaduTag → Location → "While Using."

</details>
